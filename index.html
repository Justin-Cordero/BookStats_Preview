Book Stats

The application that will parse through a book or document and give you statistics about the words used

Live Project

Need information about a book or document?

Maybe you want to improve SEO on your blog posts, or you are just curious about the most used words from your favorite book. Book Stats is an application designed to do just that. Using JavaScript, Book Stats will parse through a text document and count the most commonly used words, the least used words, and even identify the occurrences of a specific word that you search for. Additionally, a list of "stop words" were included to prevent commonly used words, like conjunctions, from affecting the data results. AJAX (Asynchronous JavaScript and XML) was used so the full document and data could be retrieved from the server and loaded into the webpage in real-time.

  • Bootstrap 4

    Bootstrap was chosen to build the UI (user interface) for Book Stats. The focus of this project was to gain a better understanding of AJAX implementation and not necessarily the UI, so Bootstrap allowed for a quick and efficient layout that looks clean and presentable.

  • JavaScript

    JavaScript was the primary technology used in the creation of the application. Specifically, I chose to build this project to work on implementing AJAX in a project, which grabbed the document from the server. Then I was able to manipulate the DOM (Document Object Model) to embed the document in the page.

    Regular Expressions (RegEx) were also used to filter the document data and eliminate line breaks, carriage returns, and punctuation from the word counts.

  • CSS

    Some standard CSS was used in this project to improve the look of the application.

    Media queries were also included to improve the responsiveness across screen sizes where Bootstrap classes were not used.

Result

There was a lot learned building this project. It is fully functional with a clean, albeit simple, user interface. In the future, I would like to improve upon this project by adding some additional features. Currently it only allows you to see the stats of books that I have already uploaded using a free-for-use eBook website (credits listed in project). Allowing users to upload their own books or documents would increase the usefulness of this application. This is something I plan on expanding upon in the future. Thanks for checking out my work.
